# ベビー用商品管理システム

## 概要

ベビー用品を扱う会社の商品の管理を行えます
 ・商品の登録・編集・削除を行うことができます
 ・従業員の登録・編集・削除を行うことができます


## 主な機能
- ログイン・ログアウト機能
- ホーム画面
- 全商品一覧画面
- 各カテゴリー（女の子・男の子・グッズ）毎の商品一覧画面
- 商品の新規登録・編集・削除
- 管理用検索枠
- 従業員一覧画面
- 従業員の新規登録・編集・削除

## 開発環境

```
PHP 8.2.0
MySQL 5.7.39
Laravel 10.13.5
```
